ROS packages for Pioneer 3 AT mobile robot
======
 
Documentation is mainly useful for students in Yamamoto laboratory. These are the packages we use.

Hardware:  
  - Pioneer 3AT
  - Hokuyo YVT-35LX   

Software:  
  - RosARIA
  - Pioneer launch and config files
  - lookahead control
  - hokuyo3d driver (VSSP 2.1)
